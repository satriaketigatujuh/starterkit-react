## React Starter Kit

Starter kit React modern dengan **Vite + TypeScript**.

### Fitur

- **Build cepat**: menggunakan Vite.
- **Type safety**: konfigurasi TypeScript ketat.
- **Struktur rapi**: entry `src/main.tsx` dan komponen utama `src/App.tsx`.

### Prasyarat

- Node.js versi terbaru LTS (disarankan \>= 18).
- npm atau pnpm atau yarn (contoh di bawah pakai npm).

### Cara install

```bash
npm install
```

### Menjalankan mode development

```bash
npm run dev
```

Lalu buka alamat yang muncul di terminal (biasanya `http://localhost:5173`).

### Build untuk produksi

```bash
npm run build
```

### Preview build produksi

```bash
npm run preview
```
